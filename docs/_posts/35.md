## 描述

> 在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组,求出这个数组中的逆序对的总数 P。并将 P 对 1000000007 取模的结果输出。 即输出 P%1000000007

示例

```
1,2,3,4,5,6,7,0

=> 7（一共有10，20，30，40，50，60，70共7对）
```
